## 快速开始

### 安装依赖

```sh
npm i
npm i rollup -g
```

### 基础实践

相关代码、配置在 basic 目录中。

进入 basic 目录执行`rollup -c`开始构建。

### 构建一个工具库

相关代码、配置在 library 目录中。

进入 library 目录执行`rollup -c`开始构建。
